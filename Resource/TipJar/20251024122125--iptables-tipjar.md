# Iptables TipJar

## Iptables TipJar Highlights

### Remove Rule from Chain
```bash
iptables -D <CHAIN_NAME> <RULE_NUMBER>
```

### List all rules with line numbers

```bash
iptables -nvL --line-numbers
```
